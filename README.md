# Oregon Town Car API

API-only Next.js application serving hardcoded JSON data for Oregon Town Car website.

## Endpoints

- `/api/services` - List of all services
- `/api/services/[slug]` - Service detail
- `/api/cities` - List of cities
- `/api/cities/[slug]` - City detail
- `/api/fleet` - Fleet vehicles
- `/api/fleet/[slug]` - Vehicle detail
- `/api/homepage` - Homepage data
- `/api/homepage-blocks` - Homepage content blocks
- `/api/seo` - SEO metadata
- `/api/navigation` - Navigation menu
- `/api/footer` - Footer data
- `/api/contact` - Contact information
- `/api/settings` - Site settings
- `/api/blog` - Blog posts
- `/api/blog/[slug]` - Blog post detail
- `/api/reviews` - Customer reviews
- `/api/promotions` - Active promotions
- `/api/service-area` - Service coverage area
- `/api/booking-url` - External booking URL
- `/api/linking-rules` - Entity relationship mappings

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy
